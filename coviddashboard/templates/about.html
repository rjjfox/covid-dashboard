About

The dashboard has been created for and by a Brit waiting to move in with his partner in France. To get an idea of what's happening in either country, I find myself checking across multiple tabs/dashboards eagerly watching daily case numbers, waiting for these waves to hit their peak.

Instead of having multiple tabs open constantly, I decided to create a single dashboard that acts as a one-stop-shop to get all the information you need to understand the situation between the UK and France.

See the Github repository if you are interested in learning more about how the dashboard was built.

Updates

  • 31 Jan 2021, Vaccination tab added - after the UK made daily vaccination information available through their API.
  • 18 Dec 2021, Data for booster jabs included

Upcoming Changes

  • News feed widget pulling out articles regarding lockdown plans, vaccination programs and more. This will likely leverage the News API
  • Add infection rate per 100,000 cases - this was previously the method by which the UK decided whether a country would be part of a travel corridor or not.
